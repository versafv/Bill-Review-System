# 后端总览

基于 **RuoYi-Vue** 的 Spring Boot 多模块项目，Maven 坐标 `com.zsc:zsc:3.9.1`。

## 模块依赖层次

```
zsc-admin（启动入口 + 所有 Controller）
  ├── zsc-framework（Security / AOP / 配置 / 基础设施）
  │     └── zsc-system（系统实体 / Mapper / Service）
  │           └── zsc-common（注解 / 工具类 / 基类 / 枚举）
  ├── zsc-quartz（Quartz 定时任务）
  │     └── zsc-common
  ├── zsc-generator（Velocity 代码生成器）
  │     └── zsc-common
  └── zsc-module（自定义业务模块 - 图书分类）
        └── zsc-framework
```

## 两种架构风格

系统内存在两套响应体系：

| 维度 | 系统模块 (zsc-admin/system) | 业务模块 (zsc-module) |
|------|--------------------------|---------------------------|
| Controller 基类 | 继承 `BaseController` | 直接 `@RestController` |
| 响应类 | `AjaxResult extends HashMap` | `ResultVo<T>` (POJO) |
| 分页方式 | `startPage()` + `getDataTable()` | `BasePageReq` / `PageResult` |
| 实体基类 | 继承 `BaseEntity` / `TreeEntity` | Lombok `@Data`，不继承基类 |
| DTO/VO | 无分离 | DTO / VO / QueryDto 严格分离 |
