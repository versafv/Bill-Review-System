# 详细设计

> 完整文档: [[../docs/详细设计说明书|详细设计说明书.md]]

## 数据库设计

### 核心表

| 表名 | 说明 |
|------|------|
| sys_user | 系统用户 |
| sys_role | 角色 |
| sys_menu | 菜单/权限 |
| sys_user_role | 用户-角色关联 |
| sys_role_menu | 角色-菜单关联 |
| biz_bill | 票据信息 |
| biz_category | 票据类别 |

### 票据表 (biz_bill)

| 字段 | 类型 | 说明 |
|------|------|------|
| id | bigint | 主键 |
| bill_no | varchar(32) | 票据编号 |
| title | varchar(255) | 票据标题 |
| category_id | bigint | 票据类别 |
| amount | decimal(10,2) | 金额 |
| description | varchar(500) | 描述 |
| status | char(1) | 状态: 0-草稿 1-已提交 2-已通过 3-已退回 |
| create_by | varchar(64) | 创建者 |
| create_time | datetime | 创建时间 |
| audit_by | varchar(64) | 审核人 |
| audit_time | datetime | 审核时间 |
| audit_comment | varchar(500) | 审核意见 |

## API 设计

### 票据相关接口

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | /biz/bill/list | 票据列表（分页+条件查询） |
| GET | /biz/bill/{id} | 票据详情 |
| POST | /biz/bill | 新增票据 |
| PUT | /biz/bill | 修改票据（仅草稿） |
| DELETE | /biz/bill/{ids} | 删除票据（仅草稿） |
| PUT | /biz/bill/audit | 审核票据 |

### 通用返回格式

```json
{
  "code": 200,
  "msg": "操作成功",
  "data": {}
}
```
