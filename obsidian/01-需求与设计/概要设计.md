# 概要设计

> 完整文档: [[../docs/概要设计说明书|概要设计说明书.md]]

## 系统架构

前后端分离架构：

```
┌─────────────────────────────────┐
│         浏览器 (Browser)         │
├─────────────────────────────────┤
│     Vue 3 + Element Plus        │
│     (SPA, 前端应用)              │
├─────────────────────────────────┤
│     HTTP / JSON (REST API)      │
├─────────────────────────────────┤
│     Spring Boot (后端)           │
│     ├── Controller 层            │
│     ├── Service 层               │
│     ├── Mapper 层 (MyBatis-Plus) │
│     └── Model 层                 │
├─────────────────────────────────┤
│     MySQL 8.0 + Redis            │
└─────────────────────────────────┘
```

## 后端模块

| 模块 | 职责 |
|------|------|
| zsc-admin | 应用入口，Controller |
| zsc-framework | 框架核心（Security、JWT、Redis） |
| zsc-system | 系统管理（用户、角色、菜单） |
| zsc-module | 业务模块（票据） |
| zsc-common | 公共工具 |
| zsc-quartz | 定时任务 |
| zsc-generator | 代码生成 |

## 技术选型理由

| 技术 | 理由 |
|------|------|
| RuoYi-Vue | 成熟的开源后台管理框架，自带 RBAC、代码生成 |
| Spring Boot 3 | 最新稳定版，支持 JDK 17 |
| MyBatis-Plus | 简化 CRUD，减少手写 SQL |
| Spring Security + JWT | 无状态认证，适合前后端分离 |
| Vue 3 + Vite | 现代化前端工具链 |
| Element Plus | 成熟的企业级 Vue 3 组件库 |
