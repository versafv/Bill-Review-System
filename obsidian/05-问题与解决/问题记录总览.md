# 问题记录总览

开发过程中遇到的 Bug 和解决方案汇总。

## 使用方式

- 新建笔记 → `Ctrl+T` → 选择"问题记录"模板
- 记录: 问题描述 → 复现步骤 → 原因分析 → 解决方案
- 解决后更新 `status` 为"已解决"

## 问题列表

```dataview
TABLE status, created
FROM "05-问题与解决"
WHERE file.name != "问题记录总览"
SORT created DESC
```

## 常见问题分类

### 环境配置
- 数据库连接失败
- Redis 连接失败
- Maven 依赖下载失败
- npm install 失败

### 后端
- 接口 401/403 错误
- JWT Token 过期
- MyBatis SQL 语法错误
- 跨域 (CORS) 问题

### 前端
- 路由跳转失败
- 组件数据不更新
- Element Plus 版本兼容
- 打包部署问题
