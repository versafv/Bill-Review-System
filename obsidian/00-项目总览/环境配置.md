# 环境配置

## 基础环境

| 工具 | 版本要求 | 说明 |
|------|----------|------|
| JDK | 17+ | 后端运行环境 |
| MySQL | 8.0+ | 数据库 |
| Redis | 6.0+ | 缓存服务 |
| Maven | 3.6+ | 后端构建 |
| Node.js | 22+ | 前端构建 |

## 数据库

### 创建数据库

```sql
CREATE DATABASE zsc CHARACTER SET utf8mb4 COLLATE utf8mb4_general_ci;
```

### 导入表结构

```bash
mysql -u root -p < zsc_backend/sql/zsc.sql
mysql -u root -p < zsc_backend/sql/biz_bill.sql
```

### 配置连接

编辑 `zsc_backend/zsc-admin/src/main/resources/application-druid.yml`：

```yaml
datasource:
  master:
    url: jdbc:mysql://localhost:3306/zsc?useUnicode=true&characterEncoding=utf8
    username: root
    password: 你的密码
```

## Redis

Windows 下可使用 [Memurai](https://www.memurai.com/) 或 WSL 运行 Redis。

```bash
# WSL 中
sudo apt install redis-server
sudo service redis-server start
```

## 后端启动

```bash
cd zsc_backend
mvn clean package -DskipTests
java -jar zsc-admin/target/zsc-admin.jar
```

- API: http://localhost:8080
- Swagger: http://localhost:8080/swagger-ui/index.html

## 前端启动

```bash
cd zsc_frontend
npm install
npm run dev
```

- 页面: http://localhost:80
