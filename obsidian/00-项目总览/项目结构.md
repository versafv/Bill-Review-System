# 项目结构

## 整体目录树

```
Bill-Review-System/
├── zsc_backend/                    # 后端 (Spring Boot 多模块)
│   ├── zsc-admin/                  # 应用入口 + Controller
│   ├── zsc-framework/              # 框架核心 (Security / Redis / JWT)
│   ├── zsc-system/                 # 系统管理 (用户 / 角色 / 菜单)
│   ├── zsc-module/                 # 业务模块 (票据管理)
│   ├── zsc-common/                 # 公共工具类
│   ├── zsc-quartz/                 # 定时任务
│   ├── zsc-generator/              # 代码生成器
│   └── sql/                        # 数据库脚本
│
├── zsc_frontend/                   # 前端 (Vue 3 SPA)
│   └── src/
│       ├── views/biz/bill/         # 票据管理页面
│       ├── api/biz/                # API 封装
│       ├── router/                 # 路由配置
│       └── store/                  # 状态管理
│
├── docs/                           # 设计文档
│   ├── 需求规格说明书.md
│   ├── 可行性分析报告.md
│   ├── 概要设计说明书.md
│   └── 详细设计说明书.md
│
├── obsidian/                       # Obsidian 知识库 (当前目录)
└── GIT_GUIDE.md                    # 小组 Git 协作指南
```

## 技术栈

| 层级 | 技术 | 版本 |
|------|------|------|
| 后端框架 | Spring Boot | 3.5.8 |
| ORM | MyBatis-Plus | 3.5.9 |
| 安全 | Spring Security + JWT | - |
| 数据库 | MySQL | 8.0 |
| 缓存 | Redis | - |
| 前端框架 | Vue 3 | 3.5 |
| 构建工具 (前端) | Vite | 5 |
| UI 组件库 | Element Plus | 2.13 |
| 状态管理 | Pinia | 3 |
| 构建工具 (后端) | Maven | 3.6+ |
