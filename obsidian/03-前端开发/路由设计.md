# 路由设计

## 路由模式

- **模式**: `createWebHistory()` (history 模式，无 `#`)
- **守卫**: `permission.js` -> `router.beforeEach()`

## 常量路由 (constantRoutes)

所有用户都能访问的路由：

| 路径 | 组件 | 说明 |
|------|------|------|
| `/login` | views/login.vue | 登录 |
| `/register` | views/register.vue | 注册 |
| `/redirect/:path(.*)*` | views/redirect/index.vue | 路由重定向 |
| `/404` | views/error/404.vue | 404 页面 |
| `/401` | views/error/401.vue | 401 页面 |
| `/index` | views/index.vue | 仪表盘首页 |
| `/user/profile` | views/system/user/profile/index.vue | 个人中心 |

## 动态路由 (dynamicRoutes)

前端定义 + 后端菜单权限联合控制：

```javascript
// router/index.js 中定义
[
  {
    path: '/system/user-auth',
    component: () => import('@/views/system/user/authRole.vue'),
    meta: { permissions: ['system:user:edit'] }
  },
  {
    path: '/system/role-auth',
    component: () => import('@/views/system/role/authUser.vue'),
    meta: { permissions: ['system:role:edit'] }
  },
  {
    path: '/system/dict-data',
    component: () => import('@/views/system/dict/data.vue'),
    meta: { permissions: ['system:dict:list'] }
  },
  {
    path: '/monitor/job-log',
    component: () => import('@/views/monitor/job/log.vue'),
    meta: { permissions: ['monitor:job:list'] }
  },
  // 代码生成器的子路由
  {
    path: '/tool/gen-edit',
    component: () => import('@/views/tool/gen/editTable.vue'),
    meta: { permissions: ['tool:gen:edit'] }
  }
]
```

## 后端菜单路由生成

```
GET /getRouters 返回菜单树 JSON:
[
  {
    "name": "System",
    "path": "/system",
    "component": "Layout",
    "meta": { "title": "系统管理", "icon": "system" },
    "children": [
      {
        "name": "User",
        "path": "user",
        "component": "system/user/index",
        "meta": { "title": "用户管理", "icon": "user" },
        "children": [
          {
            "name": "UserAdd",
            "path": "",
            "component": "",
            "meta": { "title": "新增用户" }
          }
        ]
      }
    ]
  }
]
```

## 动态路由生成算法

```javascript
// store/modules/permission.js → generateRoutes()
1. 调用 GET /getRouters 获取后端菜单树
2. 使用 import.meta.glob('@/views/**/*.vue') 动态导入所有页面组件
3. 遍历菜单树:
   - component 为空 → 使用 ParentView (占位)
   - component 存在 → loadView(view) 从 glob 缓存中匹配组件
   - Layout → 直接使用 layout/index.vue
   - InnerLink → 使用内嵌 iframe 组件
4. 调用 filterDynamicRoutes(dynamicRoutes) 合并前端定义的动态路由
5. router.addRoute() 逐个注册到路由表
```

## 路由元信息 (meta)

| 字段 | 类型 | 说明 |
|------|------|------|
| `title` | string | 页面标题 (显示在 TagsView) |
| `icon` | string | 菜单图标 (svg 或 element-icon) |
| `noCache` | boolean | 是否不缓存 (keep-alive 的白名单) |
| `link` | string | 外部链接地址 (InnerLink 模式) |
| `roles` | string[] | 允许的角色列表 |
| `permissions` | string[] | 允许的权限列表 |

## 标签页 (TagsView) 路由联动

- 路由跳转 → `tagsViewStore.addView(route)` → 新增标签
- 关闭标签 → `tagsViewStore.delView(route)` → 导航到右侧标签或首页
- 右键菜单: 刷新 / 关闭当前 / 关闭其他 / 关闭左侧 / 关闭右侧 / 关闭全部
- 缓存: `cachedViews` 联动 `keep-alive :include`
