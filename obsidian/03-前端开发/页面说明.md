# 页面说明

## 登录与认证

### 登录页 (`views/login.vue`)
- 用户名/密码/验证码三要素
- "记住我"功能：密码通过 jsencrypt RSA 加密存 Cookie
- 默认测试账号: `admin / admin123`
- 成功 → 存 token → 获取用户信息 → 生成动态路由 → 跳转首页

### 注册页 (`views/register.vue`)
- 用户名、密码、确认密码（等值校验）、验证码
- POST `/register` → 成功后弹窗提示 → 跳转登录

## 首页仪表盘 (`views/index.vue`)

4 个统计卡片 + ECharts 饼图（分类分布） + ECharts 折线图（借阅趋势） + 热门排行 + 最新入库表格。数据目前为硬编码 Mock。

## 业务模块

### 业务类别管理 (`views/biz/bizCategory/`)

4 个组件构成的标准 CRUD 页面：

| 组件 | 功能 |
|------|------|
| `index.vue` | 组装 SearchForm + CategoryTable + CategoryForm，管理状态生命周期 |
| `components/SearchForm.vue` | 搜索: categoryName + status (字典下拉) |
| `components/CategoryTable.vue` | 表格: ID, 名称, 排序, 状态(dict-tag), 时间, 操作按钮 |
| `components/CategoryForm.vue` | 弹窗表单: 名称(input), 排序(input-number 0-9999), 状态(radio) |

权限控制: `v-hasPermi` 控制新增/编辑/删除按钮可见性。

## 全局指令

| 指令 | 用法 | 说明 |
|------|------|------|
| `v-hasPermi` | `v-hasPermi="['system:user:add']"` | 无权限时移除 DOM 元素，`*:*:*` 超级管理员绕过 |
| `v-hasRole` | `v-hasRole="['admin']"` | 无角色时移除 DOM 元素，`admin` 角色绕过 |
| `v-copyText` | `v-copyText="text"` | 点击复制文本到剪贴板 |
